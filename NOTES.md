# Neovim Tips

## Neo-tree (File Explorer)

**Toggle:** `Ctrl+B` — opens tree and reveals current file in context

### Navigation
- `o` or `Enter` — open file
- `P` — preview file without leaving the tree
- `<` / `>` — navigate up/down to parent/sibling directories
- `H` — toggle hidden files
- `/` — search files in the current tree
- `f` — filter files live as you type
- `R` — refresh the tree

### File Operations
- `a` — create new file
- `d` — delete
- `r` — rename
- `m` — move
- `c` — copy

### Opening Files
- `bs` — open in horizontal split
- `bd` — open in vertical split

## Window Management

- `Ctrl+W Ctrl+W` — cycle focus between windows/splits
- `Ctrl+W h` / `Ctrl+W l` — move left/right between windows
- `Ctrl+W q` — close current window/split
